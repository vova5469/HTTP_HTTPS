HTTP методы:
GET - запрашивает представление ресурса.
HEAD - запрашивает ресурс так же, как GET, но без тела ответа.
POST - используется для отправки данных к определенному ресурсу.
PUT - служит для изменения или вставки ресурса.
DELETE - удаляет указаный ресурс.
OPTIONS - используется для описания параметров коммуникации между клиентом и сервером.
CONNECT - устанавливает "тунель" к серверу, определенному по ресурсу.
TRACE - выполняет вызов возвращаемого текстового сообщения с ресурса.
PATCH - используется для частичного изменения ресурса.

Идемпотентные HTTP методы:
DELETE, GET, HEAD, OPTIONS, PUT, TRACE

Безопасные HTTP методы:
GET, HEAD, OPTIONS, TRACE
