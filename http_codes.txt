HTTP статус коды сервера:

Информационные:
100-продолжить
101-переключение протокола
102-в обработке
103-ранние подсказки

Успешные:
200-успешно
201-создано
202-принято
203-информация не авторитетна
204-нет содержимого
205-сбросить содержимое
206-частичное содержимое

Сообщения о перенаправлениях:
300-множественный выбор
301-перемещен на постоянной основе
302-найдено(временно изменен)
303-просмотр других ресурсов
304-не модифицировано
305-использовать прокси
306-больше не использовать
307-временное перенаправление
308-перенаправление на постоянной основе

Клиентские:
400-плохой запрос
401-неавторизовано
402-необходима оплата
403-запрещено(нет прав доступа)
404-не найден
405-метод не разрешен
406-не нашел контента, отвечающего критериям
407-аналог 401, только аутенфикация для прокси сервера
408-время ожидания сервером истекло
409-конфликт запроса с текущим состоянием сервера
410-запрашиваемый контент удален с сервера
411-сервер требует указание заголовка Content-Length
412-клиент указал в заголовке условия, которые сервер не может выполнить
413-размер запроса превышает лимит
414-URl запрашиваемый клиентом слишком длинный для того, чтобы сервер смог его обработать
415-медиа формат запрашиваемых данных не поддерживается сервером
416-сервер не может обслуживать запрошеные диапазоны
417-ожидание, полученое из заголовка запроса Expect, не может быть выполнено сервером

Серверные:
500-внутренняя ошибка сервера
501-не выполнено(метод запроса не поддерживается сервером)
502-плохой шлюз
503-сервер не доступен
504-сервер, работающий в качестве шлюза или прокси, не получил в установленый промежуток времени ответ от вышестоящего сервера, к которому он обратился
505-HTTP-версия не поддерживается
