HTTP - протокол прикладного уровня передачи данных изначально - в виде гипертекстовых документов в формате HTML, в настоящее время используется для передачи произвольных 
данных.
HTTP является протоколом клиент-серверного взаимодействия, что означает инициирования запросов к серверу самим получателем, обычно веб-браузером.

HTTPS - расширение протокола HTTP для поддержки шифрования в целях повышения безопасности.
Данные в протоколе HTTPS передаются поверх криптографических протоколов TLS/SSL
